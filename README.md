# API de growdevers

Essa api faz parte do módulo de node.js / express da plataforma de treinamento da grow dev, nesse projeto nós aprendemos a realizar um CRUD simples utilizando express.

O deploy da aplicação foi realizado na plataforma render e documentação da API está no link abaixo:

<strong>Link do deploy da API utilizando render</strong>
<a>https://academy-growdevers-api.onrender.com</a>


<strong>Link da documentação da API</strong>
<a>https://documenter.getpostman.com/view/25882412/2sBXVoA8c7</a>

Para clonar esse repositório utilize os seguintes comandos:

```
git clone https://github.com/avbds002/modulo-de-express-growDev.git
```

Para instalar as dependências do projeto e configurar as variáveis de ambiente siga o módelo fornecido no arquivo .env.example e utilize os seguintes comandos após clonar o projeto

```
npm install
npm run dev -> para rodar em modo desenvolvimento
npm run start -> para rodar em produção
```
